Your task is to implement a Binary Search Tree and verify that it correctly implements a mathematical Set in Dafny. The tree must maintain the BST Invariant: for any node with value $v$, all values in the left subtree are less than $v$ and all values in the right subtree are greater than $v$. Here, you just need to focus on the search operation. Search: Prove that the method returns true if and only if the target value is a member of the mathematical set represented by the tree's view. Note that you don't need to consider the balancing and rotation of a BST, and you can implement the naive BST.
