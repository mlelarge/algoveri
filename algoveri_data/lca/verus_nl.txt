Your task is to implement the Lowest Common Ancestor (LCA) algorithm for a binary tree and verify its correctness in Verus. Given a binary tree and two values `p` and `q` that exist in the tree, find the lowest node (deepest level) that has both `p` and `q` as descendants (where we allow a node to be a descendant of itself). In the incomplete code, the specification defines the tree structure and the `is_lca` property. Your task is to implement the recursive search algorithm and prove: (1) Correctness: The returned value is indeed the unique LCA of `p` and `q`. (2) Existence: Since `p` and `q` are guaranteed to exist, a valid LCA must exist.
