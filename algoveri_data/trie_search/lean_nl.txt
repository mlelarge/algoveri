Your task is to implement the Search operation for a Byte-based Trie data structure (Prefix Tree) and verify its correctness in Lean. The Trie represents keys as lists of bytes (integers 0-255). Postconditions: Functional Correctness (returns true iff key exists in the tree's view). Preconditions: Operation assumes the input tree satisfies well-formedness invariants (children array size 256, no useless nodes).
