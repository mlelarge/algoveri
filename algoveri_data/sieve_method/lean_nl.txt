Your task is to implement the Sieve of Eratosthenes algorithm and verify its functional correctness in Lean. Given a non-negative integer n, return a boolean array of size n where the i-th element is true iff i is prime. Postconditions: result[i] is true iff i is prime (i > 1 and no divisors). Preconditions: n is a natural number (u64 simulated). Optimization: Use the standard sieve process: start with all true, iterate p from 2 to sqrt(n), if p is prime, mark all multiples of p starting from p*p as false.
