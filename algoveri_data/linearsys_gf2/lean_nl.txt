Your task is to implement a Gaussian Elimination solver for linear systems over GF(2) and verify its correctness in Lean. Given a binary matrix A (m x n) and a binary vector b (length m), find a binary vector x (length n) such that Ax = b (modulo 2), or return None if no solution exists. Postconditions: If result is Some(x), then Ax = b (mod 2). If result is None, then no such x exists. Preconditions: Matrix and vector dimensions match. Elements are 0 or 1. Implementation should use standard Gaussian elimination with row operations modulo 2.
